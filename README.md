# CB2 Toolkit

A multi-functional toolkit for developing, building, and managing modifications. The application provides a user-friendly graphical interface for your workflow.

## ✨ Features

* **AngelScript Editor** - A specialized editor for writing and editing game scripts.
* **Addon Editor** - A dedicated tool for creating and packaging addons.
* **UI Editor** - A module for designing and configuring user interfaces *(under development)*.

## 💻 System Requirements

* **Operating System:** Windows only (Windows 10 / 11). Linux and macOS are currently not supported.

## 🚀 How to Run

1. Go to the **[Releases](https://github.com/PavlikHTP/CB2-Toolkit/releases)** section on GitHub.
2. Download the latest release archive (e.g., `CB2Toolkit_vX.X.X.zip`).
3. Extract the archive to any folder and run `CB2Toolkit.exe`.

## 🐛 Troubleshooting & Crashes

If the application crashes or encounters a critical error, it automatically generates a detailed report file. To help us identify and fix the bug, please **open a new GitHub Issue** and attach the crash data.

* **Crash Logs Location:** Check the `CrashLogs` folder inside the application's AppData directory.
* **Filename format:** `crash_YYYY-MM-DD_HHMMSS.txt`

> ⚠️ **Important:** Please copy and paste the *entire* content of the `.txt` file into your issue description, including the system environment specifications, exception info, and the full `StackTrace`.
